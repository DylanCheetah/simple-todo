# Lesson 01 - Prerequisites

In this tutorial series, we will learn full-stack web development by creating a simple todo list web app. 
Before you begin, you will need to install the following software:
* Node.js
* Python
* Visual Studio Code

On Linux you will need to also install the `python3-venv` package via your package manager.

Create a folder for your project and open it in Visual Studio Code. Click Terminal > New Terminal to open
a new terminal in Visual Studio Code. Then execute the following command to create a Python virtual 
environment:
```sh
python -m venv venv
```

On Windows, execute the following commands to activate the virtual environment:
```sh
cmd
venv\Scripts\activate
```

On Linux, execute the following command to activate the virtual environment:
```sh
./venv/bin/activate
```
