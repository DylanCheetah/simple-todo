# Lesson 14: Setup Bootstrap

Now that we have our React app setup, let's setup Bootstrap to help style our frontend. Copy the `bootstrap.css` and `bootstrap.bundle.js` files from the Bootstrap .zip file we extracted in lesson 10 to `simple-todo-frontend/src/`.

Open `simple-todo-frontend/src/index.js` and add the following imports:
```js
import "./bootstrap.css";
import "./bootstrap.bundle";
```
