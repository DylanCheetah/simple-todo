# Lesson 01: Prerequisites

In this tutorial series we will be creating a simple todo list website and mobile app. Before we begin we will need to install the development tools we will be using:
01. install Visual Studio Code from https://code.visualstudio.com/download
02. start Visual Studio Code
03. click the Extensions tab
04. search for "python"
05. install the Python extension provided by Microsoft
06. search for "kivy"
07. install the Material Kivy extension provided by haddie bakrie
08. install Python
    * Windows: visit https://www.python.org/downloads/
    * Linux: install/update via your package manager
    * MacOS: visit https://www.python.org/downloads/
