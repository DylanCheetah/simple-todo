# Lesson 14: Progressive Web App - Prerequisites

Now that we have fully implemented our backend and web interface, let's explore how to develop progressive web apps. Most progressive web apps consist of a single webpage with a frontend developed with a frontend framework such as React or with plain JavaScript. In this tutorial we will stick to using just JavaScript in order to avoid adding extra dependencies. Before we begin we will need to modify our `requirements.txt` file like this:
```
Django
djangorestframework
djangorestframework-simplejwt
```

Make sure your virtual environment is active and then execute the following command to install the new dependencies:
```sh
pip install -r requirments.txt
```
